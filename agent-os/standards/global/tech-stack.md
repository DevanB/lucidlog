## Tech stack

Define your technical stack below. This serves as a reference for all team members and helps maintain consistency across the project.

### Framework & Runtime

- **Application Framework:** Laravel
- **Language/Runtime:** PHP
- **Package Manager:** composer

### Frontend

- **JavaScript Framework:** React
- **CSS Framework:** Tailwind CSS
- **UI Components:** shadcn/ui

### Database & Storage

- **Database:** PostgreSQL
- **ORM/Query Builder:** Eloquent
- **Caching:** Redis

### Testing & Quality

- **Test Framework:** Pest (<https://pestphp.com>)
- **Linting/Formatting:** Pint (<https://laravel.com/docs/12.x/pint>) and Ultracite (<https://www.ultracite.ai>)

### Deployment & Infrastructure

- **Hosting:** Laravel Cloud (<https://cloud.laravel.com>)
- **CI/CD:** GitHub Actions

### Third-Party Services

- **Authentication:** Internal Laravel Auth
- **Email:** Resend (<https://resend.com>)
- **Monitoring:** Nightwatch (<https://nightwatch.laravel.com>)
