## Tech stack

Define your technical stack below. This serves as a reference for all team members and helps maintain consistency across the project.

### Framework & Runtime

- **Application Framework:** Laravel
- **Language/Runtime:** PHP
- **Package Manager:** composer

### Frontend

- **JavaScript Framework:** React
- **CSS Framework:** Tailwind CSS
- **UI Components:** shadcn/ui

### Database & Storage

- **Database:** PostgreSQL
- **ORM/Query Builder:** Eloquent
- **Caching:** Redis

### Testing & Quality

- **Test Framework:** Pest (<https://pestphp.com>)
- **PHP Formatting:** Laravel Pint (<https://laravel.com/docs/12.x/pint>)
- **PHP Static Analysis:** Larastan (<https://github.com/larastan/larastan>)
- **PHP Refactoring:** Rector (<https://getrector.com>)
- **JS/TS Linting & Formatting:** Ultracite (<https://www.ultracite.ai>)
- **Git Hooks:** Lefthook (<https://github.com/evilmartians/lefthook>)
- **Code Reviews:** CodeRabbit CLI (<https://www.coderabbit.ai>)

### Deployment & Infrastructure

- **Hosting:** Laravel Cloud (<https://cloud.laravel.com>)
- **CI/CD:** GitHub Actions

### Development Tools

- **Local Environment:** Laravel Herd (<https://herd.laravel.com>)
- **Route Generation:** Laravel Wayfinder (<https://wayfinder.laravel.com>)
- **Debugging:** Laravel Telescope + Laravel DebugBar
- **MCP Server:** Laravel Boost (<https://mcp.laravel.com>)

### Third-Party Services

- **Authentication:** Laravel Fortify
- **Email:** Resend (<https://resend.com>)
- **Queue Monitoring:** Laravel Nightwatch (<https://nightwatch.laravel.com>)
- **Error Tracking:** Sentry (<https://sentry.io>)
